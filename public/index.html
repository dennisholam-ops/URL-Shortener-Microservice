URL Shortener Microservice

Shorten a URL

Example Usage

POST to /api/shorturl

Body: {"url":"https://www.freecodecamp.org"}

Response: {"original_url":"https://www.freecodecamp.org","short_url":1}

Usage

Visit: /api/shorturl/<short_url> to be redirected to the original URL.

Example: /api/shorturl/1

What it does

  • Accepts a URL via POST request
  • Returns a JSON with original URL and short URL
  • Redirects short URLs to original URLs
  • Validates URL format
