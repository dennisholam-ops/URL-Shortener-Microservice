URL Shortener Microservice

How to Use:

  1. POST a URL to [project_url]/api/shorturl
  2. Get a JSON response with original_url and short_url
  3. Visit [project_url]/api/shorturl/<short_url> to be redirected

Shorten a URL

Example Usage:

POST Request:

POST /api/shorturl

Body: url=https://www.freecodecamp.org

Response:

{ "original_url": "https://www.freecodecamp.org", "short_url": 1 }

Usage:

Visit: /api/shorturl/1 to redirect to freeCodeCamp
