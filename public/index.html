URL Shortener Microservice

Short URL Creation

Example: POST [project_url]/api/shorturl - https://www.google.com

URL Shortener

Example Usage:

[this_project_url]/api/shorturl/1

Will redirect to: https://www.freecodecamp.org/

How It Works:

  1. POST a valid URL to /api/shorturl
  2. Receive a JSON response with original_url and short_url properties
  3. Visit /api/shorturl/<short_url> to be redirected to the original URL
