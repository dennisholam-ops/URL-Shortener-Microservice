URL Shortener Microservice

URL Shortener

Example Usage:

POST to [project_url]/api/shorturl with body: {"url":"https://www.freecodecamp.org"}

Returns: {"original_url":"https://www.freecodecamp.org","short_url":1}

Then visit: [project_url]/api/shorturl/1 to be redirected to the original URL
